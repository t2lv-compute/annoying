Annoying Tkinter Window

DANGER: This app can potentially block up your entire screen, use up crazy levels of computing power. If this happens, force restart your laptop. USE WITH CAUTION.

This is an annoying app. When you open it, you will see a window with two buttons. The window will drift around. The buttons say "Duplicate Window" and "Press this to explode" or something similar. If you try to close the window, it will immediately reopen. If you press Duplicate Window, it will duplicate the window. Both windows will now drift around. To close the windows, press the "Press this to explode" button and click on every grey square, turning them red. Once this is complete, all windows will close.

The .exe version of the app is located in the /dist folder. You can also run the Python version from john-annoyance.py. If you run the app from the Python folder, you can press Ctrl-C in the terminal until all windows close.

IMPORTANT: As a backdoor in case things get out of hand, you can press Shift-Esc to close all windows in one go. If you can, you can also use the Task Manager or similar application to stop the app.
